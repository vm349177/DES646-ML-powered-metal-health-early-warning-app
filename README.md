# DES646-ML-powered-metal-health-early-warning-app

## How to install

create a .venv (used python 3.10)

activate .venv
```bash
.venv/Scripts/activate
```

install all requirements
```bash
pip install -r requirements.txt
```

Manually add the unzipped file to output directory (output_fast_model)(max file size se badha tha sorry)

then run voice_sentiment.py
```bash
python voice_sentiment.py
```
